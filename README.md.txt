#  Oil and Gas Vendor Performance Analysis

A data analysis project focused on evaluating vendor performance in the oil and gas sector using Python.
This project analyzes cost efficiency, quality performance, and contract trends to generate actionable business insights.


## Project Overview

The dataset contains vendor-level operational data, including:

* Contract types
* Cost variance
* Quality scores
* Purchase order activity

The goal is to assess vendor performance and identify patterns that impact operational efficiency.

---

## Tech Stack

* **Python**
* **Pandas**
* **NumPy** 
* **Matplotlib** 
* **Jupyter Notebook**

---

## Analysis Performed

### 1. Vendor Performance Analysis

* Evaluated cost variance across vendors
* Measured quality scores
* Analyzed purchase order frequency

### 2. Contract Type Insights

* Compared performance across:

  * Fixed Price
  * Time & Materials
* Identified cost efficiency trends

### 3. Category Analysis

* Grouped vendors by category
* Measured defect rates
* Identified high-risk vendor segments

---

## Visualizations

The project includes:

* Vendor performance comparisons
* Contract type analysis
* Category-based defect rate insights

---

## Key Insights

* Vendors under **Fixed Price contracts** show more predictable cost behavior
* Certain categories exhibit higher defect rates, indicating quality risks
* High-performing vendors maintain both low cost variance and high quality scores
